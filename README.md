<h1>Projeto Final de Inteligência Artificial</h1>

<h3>Segurança operacional (rotas de evacuação/lockout-tagout)</h3>

O código principal esta no "main.py".

Para fazer a busca é preciso:
- Escolher um método de busca do menu
- Clicar no botão "Escolher a célula de inicio" e clicar em uma célula no grid
- Clicar no botão "Escolher a célula do fim" e clicar em uma ou mais células no grid
- E depois clicar em "Fazer busca" para realizar a busca

Código feito com PyCharm e TKinter foi usado.

**Programa feito por Paulo Vilalta e Pedro Henrique Santos.**

> [!NOTE]
> Os algoritmos de busca em grid disponíveis são:<br>
> AMPLITUDE, PROFUNDIDADE, PROFUNDIDADE LIMITADA, APROFUNDAMENTO ITERATIVO, BIDIRECIONAL, CUSTO UNIFORME, GREEDY, A* e AIA*.
